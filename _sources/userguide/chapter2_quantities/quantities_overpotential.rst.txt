Overpotential
------------------
The **overpotential** is the deviation from the equilibrium potential required to drive an electrochemical reaction.

**Example:** Representing an overpotential of 0.2V:

.. code-block:: json

   {
     "@context": "https://w3id.org/emmo/domain/electrochemistry/context",
     "@type": "Overpotential",
     "hasNumericalPart": {
       "@type": "RealData",
       "hasNumberValue": 0.2
     },
     "hasMeasurementUnit": "emmo:Volt"
   }
